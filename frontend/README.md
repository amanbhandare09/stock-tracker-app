## Teck Stack
- Next.JS
- Tailwind CSS
- DaisyUI
- ShadCN



## Getting Started
First, to run the development server:
cd into the frontend directory
```bash
cd frontend
```
Install the dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```
Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
